# 🎭 一致性檢查 (Consistency Check) Prompt

請根據目前的章節草稿 `chXXX.md`，比對 `novel_01/bible/` 中的設定，確認以下重點：

1. **邏輯衝突**: 是否有之前設定過不行，現在卻可以的事情？
2. **性格跑偏**: 主角與配角的說話口吻是否與 `characters.yaml` 的 `speech_style` 一致？
3. **專有名詞**: 確認地名、術語、物品名稱的拼寫或翻譯是否統一。
4. **時間線**: 檢查 `timeline.yaml` 是否漏掉了本章的時間跨度。

請列出所有潛在的不一致之處，並給出修改建議。
