# 深淵地下城群像劇

- `Novel ID`: `novel_04_dungen`
- `Status`: worldbuilding
- `Current Date`: `深淵曆 Y000-M01-D01`
- `Project Folder`: `novel_04_dungen`

本專案採用「既有小說專案格式 + 深淵活世界更新」的混合架構。`fantasy_world_master_outline.md` 是完整母綱，保存世界設計源頭；日常寫作前優先閱讀 `bible/worldbuilding.md`、`bible/character.md` 與 `updates/current_status.md`。

## 核心定位

這是一個以城市文明為秩序表層、以深淵型地下遺跡為資源與未知核心、以冒險者／傭兵活動圈作為流動中介的多種族奇幻群像劇。

故事重點不只是單一主角升級，而是城市、深淵、勢力、冒險者、研究者、商人與多種族社群在同一座前線城市中的同步變化。

## 目錄結構

- `fantasy_world_master_outline.md`: 世界觀完整母綱，作為最高設計基準。
- `calendar.md`: 輕量城市曆法與日期記錄規則。
- `bible/`: 寫作前快速注入的穩定公開 canon。
  - `worldbuilding.md`: 世界觀總覽。
  - `character.md`: 群像角色總檔。
- `regions/`: 城市大區與深淵層域設定，不記每日變動。
  - `region_index.md`: 區域索引。
  - `city_overview.md`: 特例城市總覽。
  - `abyss_overview.md`: 深淵遺跡總覽。
- `updates/`: 劇中日期動態，專門追蹤「今天發生什麼」。
  - `current_status.md`: 最新世界狀態快照。
  - `daily/`: 按劇中日期歸檔的每日動態。
- `factions/`: 勢力狀態與權力關係。
- `context/`: 寫作前 AI 注入摘要、上一章摘要、作者後台秘密。
  - `CONTEXT.md`: 長篇連貫性核心摘要。
  - `last-chapter-summary.md`: 最近章節壓縮摘要。
  - `secrets-lockbox.md`: 作者後台秘密與未揭露真相。

## 維護規則

- 母綱只在根本設定改變時修改。
- `bible/worldbuilding.md` 只放公開且穩定的世界觀總覽。
- `bible/character.md` 集中管理角色定位、狀態、目標與秘密引用。
- `regions/` 記區域長期設定，`updates/` 記日期變動。
- 未揭露真相、深淵核心秘密、勢力黑箱與角色不知道的情報放入 `context/secrets-lockbox.md`。
- 公開檔可引用秘密編號，但不要直接寫出後台真相。

## 寫作前建議閱讀順序

1. `updates/current_status.md`
2. `context/CONTEXT.md`
3. `context/last-chapter-summary.md`
4. `bible/character.md`
5. `regions/region_index.md`
6. `bible/worldbuilding.md`
7. 必要時回查 `fantasy_world_master_outline.md`
