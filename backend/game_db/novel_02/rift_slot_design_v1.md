# 次元滲透裂縫（Rift Slot）機制設計 v1

> **設計版本**：v1.0
> **建立日期**：2026-05-15
> **對應規則版本**：card_game_design v3.3（v3.2 升級）
> **狀態**：Stage 1 設計文件 — 規則與卡牌條目

---

## 1. 概述與設計目標

### 1.1 問題陳述

在 v3.2 規則中，「次元壁穩定度」是一個**單向資源**：

- 玩家想盡辦法**維持穩定度高**（避免敗北、避免階段懲罰）
- 沒有任何讓玩家「主動降低穩定度」的動機
- 結果是穩定度只是一個失敗計時器，缺乏戰術張力

### 1.2 設計目標 — 雙刃資源

將穩定度升級為**雙刃戰術資源**：當穩定度跨越 50 線後，戰場中央生成一個「次元滲透裂縫」欄位，敵我皆可利用。玩家因此面對方向相反的兩條戰略路線：

- **守序派**：努力把穩定度維持在 ≥ 50（保安全、不開裂縫）
- **混沌派**：主動讓穩定度跌破 50，搶在敵方滲透前佔據裂縫獲取強力 buff

裂縫一旦開啟即**不可逆**——這呼應 novel_02 世界觀中「次元壁表面癒合、內部從未穩定」的設定，也讓「降穩定度」的決策具備真實後果。

### 1.3 與世界觀的契合

依 `bible/worldbuilding.md`：

- **核心威脅**：「黑暗次元已開始滲透，次元壁裂縫正在擴大」（核心前提）
- **歷史背景**：「眾神之戰造成大陸分裂、多處次元壁破裂，魔族由此誕生」
- **現況**：「次元壁在眾神之戰後表面癒合，內部從未穩定，現已進入滲透初期，異常事件頻率正在上升」
- **節日鉤點**：神寂日「裂痕日」習俗——「在裂痕日前夕修繕家中破損的器物，象徵『不讓裂痕過夜』」可作 S08 次元修補的新 flavor

裂縫機制讓玩家在每一場戰鬥中**親身體驗**這種威脅——「次元壁從未穩定」不再只是世界觀台詞，而是每場戰鬥都可能成真的戰場現實。

## 2. 規則總則

### 2.1 觸發條件

- **開啟閾值**：`stability < 50`（**首次**跨越 50 線時觸發）
- **觸發時機**：在 `applyStabilityDelta` 結算後立即判定；同回合內無論再降多少都只開啟 1 個裂縫
- **不可逆性**：裂縫一旦開啟即持續至戰鬥結束。**穩定度回升至 50 以上不會關閉裂縫**——對應「裂痕一旦撕開就難以彌合」的設定
- **內戰模式停用**：與 v3.2「內戰模式停用穩定度」一致，本機制在 PvP 完全停用

### 2.2 場上數量

- 場上**永遠至多 1 個**裂縫
- 穩定度更低（< 25 等）**不會**追加裂縫——避免規則與 UI 複雜化
- 任何卡牌效果若試圖「再開一個裂縫」，視為對既有裂縫的強化（具體效果待新卡定義時細化）

### 2.3 位置與佔位

- 裂縫位於**戰場中央獨立欄位**，不佔用敵我各自的 5 個兵力欄
- 佔據裂縫的單位**不計入**雙方兵力欄上限
- 場地卡與裂縫**獨立共存**：場地卡仍照 v3.2 限 1 規則運作，與裂縫互不干涉
- 視覺上裂縫位於玩家側與敵方側兵力欄之間的「無人地帶」（Stage 3 UI 規格）

### 2.4 擁有權狀態

裂縫在任一時間點處於以下三種狀態之一：

| 狀態 | 條件 | 視覺意象 |
|------|------|----------|
| **空置（Open）** | 裂縫剛開啟、佔據者陣亡後 | 紫黑色裂口持續流溢黑暗能量 |
| **玩家佔據（Player-held）** | 玩家兵力進駐裂縫位 | 兵力被裂縫能量包裹，呈現半透紫色光暈 |
| **敵方佔據（Enemy-held）** | 敵方滲透體進駐裂縫位 | 黑暗能量結晶，伴隨腐化粒子 |

**狀態轉換規則**：

```
空置 ──玩家部署兵力選裂縫位──▶ 玩家佔據
空置 ──滲透倒數歸零──▶ 敵方佔據
玩家佔據 ──佔據者陣亡──▶ 空置（裂縫重啟滲透倒數）
敵方佔據 ──佔據者陣亡──▶ 空置（裂縫重啟滲透倒數）
```

> 佔據期間另一方無法搶位（不能「跨入」對方佔據中的裂縫）；必須先擊殺佔據者讓裂縫回到空置狀態。

## 3. 敵方滲透

### 3.1 「裂縫震動」預告機制

不依賴 AI 意圖系統，預告改由**裂縫狀態自身**驅動：

| 時點 | 事件 |
|------|------|
| 裂縫開啟當回合（回合結束時） | 顯示「**裂縫震動**」視覺強告示與敘事字幕；倒數設為 1 |
| 倒數每回合結束 -1 | （無中間階段——倒數直接從 1 進入 0） |
| 倒數歸 0（下回合結束時） | **滲透**——選定 1 張滲透體出現在裂縫位（敵方佔據）；倒數重設為 1 |
| 裂縫被玩家佔據 | 倒數**暫停**；佔據者陣亡 → 倒數重設為 1 重新震動 |
| 滲透體陣亡 | 裂縫回到空置；倒數重設為 1 重新震動 |

> **節奏總結**：裂縫開啟當回合震動告示，**下一回合結束**立即滲透。一回合準備期，緊湊節奏迫使玩家立刻決策「擋下還是佔據」。

**視覺/敘事規格**（給 Stage 3 UI 參考）：
- 裂縫剛開啟：紫黑色裂口劇烈震動、邊緣冒出黑暗能量粒子；播放「**次元震動**」對話框文字
- 滲透瞬間（下回合結束）：紫黑爆閃、滲透體淡入；播放敵方滲透體名稱字幕

### 3.2 滲透體選擇邏輯

- 滲透體從 §3.3「魔族滲透體白名單」（M01–M06）中抽取
- 選擇規則：依**當前回合數**決定可用池
  - 回合 1–4：M01, M02（低費基層滲透）
  - 回合 5–8：M01–M04（加入古魔與夢魔）
  - 回合 9+：M01–M06 全池
- 同一場戰鬥中相同滲透體最多出現 2 次（避免重複堆疊）
- 具體加權與 Personality 連動的微調留到 Stage 2

### 3.4 擊殺獎勵

- 擊殺敵方滲透體（無論其是否在裂縫位上）：**玩家 +10 鬥志**
- 擊殺**不**回穩定度（保持 §2.1 不可逆原則）
- 擊殺**不**抽牌、**不**回魔力（避免「擊殺農場」化）
- 鬥志獎勵與既有「擊殺兵力 +15」**互斥**（滲透體擊殺只給 +10 滲透獎勵，不再疊加 +15 一般擊殺）

## 4. 玩家佔據

### 4.1 操作流程

- 玩家部署兵力時，可選擇將其放入「**裂縫位**」而非自家兵力欄
- 只能在裂縫處於 **Open（空置）** 狀態時佔據；敵方佔據中時必須先擊殺對方
- 佔據時兵力卡的所有正常費用、入場曲、關鍵字**照常生效**
- 不需要特殊法術或裝備門檻——任何兵力卡都可佔據（含通用、種族、中立傳說）

**UI 規格**（Stage 3 參考）：
- 部署選擇欄時，裂縫位以特殊紫色高亮顯示
- Hover 顯示「將獲得 ATK×2、DEF+5」提示

### 4.2 加乘 buff

佔據單位在**進入裂縫位的瞬間**獲得 modifier：

| 屬性 | 加乘 |
|------|------|
| ATK | × 2 |
| DEF | + 5 |
| HP | 不變 |

**Modifier 規則**：

- 進入裂縫位時掛上 `rift_occupant` modifier
- 離開裂縫位（陣亡、被位移卡牌移走、被敵方佔據體取代）時移除 modifier
- 此 modifier **不疊加**：同一單位多次進出仍只有一份效果
- 與其他 modifier 共存且**獨立計算**：如裝備帶 +ATK 與裂縫加乘併存，先套裂縫 ATK×2，再加裝備加值
- 敵方滲透體進入裂縫位同樣獲得此 modifier（對稱規則，見 §3.3 註）

> HP 不變的設計考量：避免「佔據→補滿 HP→拉鋸戰過於拖長」；ATK×2 與 DEF+5 已足夠強化但不破壞傷害經濟。

### 4.3 拉鋸戰機制

- 佔據者陣亡 → 裂縫立即回到 **Open** 狀態
- 裂縫震動倒數**重設為 2** 重新計時——敵方可以再次滲透
- 玩家也可以再次佔據——只要敵方還沒在新一輪倒數結束前滲透
- 玩家佔據期間敵方滲透**完全停止**（倒數暫停）；反之若敵方先佔據，玩家也無法直接「跨進去」搶位

**戰術含義**：
- 若玩家率先佔據，可長期關閉敵方滲透（強勢但需保住佔據者）
- 若敵方先滲透，玩家擊殺後**可立刻在自己回合再部署佔據**——擊殺與佔據之間不存在強制冷卻
- 高 HP 兵力（如重坦類）放裂縫位拉鋸價值最高，但放在裂縫的兵力等於離開了一般兵力欄陣型

### 5.1 S15 裂痕召喚（主動佔據法術）

| 欄位 | 內容 |
|------|------|
| **ID** | S15 |
| **名稱** | 裂痕召喚 |
| **類型** | 法術（紫邊框） |
| **費用** | 5 |

## 6. 既有卡牌調整

### 6.1 F08 次元裂縫【重作】

**原條目**（v3.2 / 卡表 v1）：

```
F08 次元裂縫 | 場地 | 5 費 | ★ 傳說 | 次元壁穩定度每回合 -5。但所有法術和行動卡傷害 +50%。
```

**重作後條目**（v3.3 / 卡表 v1.1）：

| 欄位 | 內容 |
|------|------|
| **ID** | F08 |
| **名稱** | 次元裂縫 |
| **類型** | 場地（靛邊框） |
| **費用** | 5 |
| **稀有度** | ★（傳說） |
| **效果** | **唯有場上已開啟裂縫時可施放**。將該裂縫升級為「**加強裂縫**」狀態，持續至戰鬥結束：<br>① 佔據者額外獲得 **〔穿透〕** 關鍵字（攻擊無視 DEF）<br>② 敵方滲透節奏不變，但**滲透體抽取池往上一階**（若當前回合應抽 M01–M02，改抽 M03–M04；M03–M04 段改抽 M05–M06；全池段不變）<br>③ 所有**法術與行動卡**傷害 **+50%**（保留 v3.2 特性，呼應「裂縫共鳴」的能量場域） |
| **Flavor** | 「裂縫從來不是裂縫，它只是還沒打開的門。」 |

**設計理由**：

- v3.2 F08 的「-5 穩定度/回合」效果**與裂縫機制衝突**（穩定度不可逆原則下，主動下降已由開啟裂縫達成；重複的降穩定度價值降低）
- 重作後 F08 成為「**已開裂縫的賭注強化牌**」——玩家獲得更強佔據 buff，但敵方滲透強度也升級
- 這完美呼應「裂縫=雙刃資源」的核心理念：你不能單方面強化它
- **不可堆疊**：場上同時只能有 1 張場地卡（v3.2 規則），F08 與其他場地互斥；同一裂縫上 F08 也只能升級 1 次

### 6.2 S08 次元修補

**原條目**：`S08 次元修補 | 3 費 | ●● | 次元壁穩定度 +15`

**v3.3 調整**：

- **效果不變**：次元壁穩定度 +15
- **新增說明文字**：「此卡無法關閉已開啟的裂縫。」
- **新 Flavor**：「她在裂痕日前夕修補一隻碗，動作熟練得像在向眾神道歉。」（呼應 worldbuilding「裂痕日」習俗「不讓裂痕過夜」）

> S08 仍然是強力的「守序派」防禦牌——避免穩定度跌破 50（避免裂縫首次開啟），但一旦裂縫已開，S08 無法回頭。這加強「跨越 50 線是不可逆決定」的緊張感。

### 6.3 N05 次元壁碎片

**原條目**：`N05 次元壁碎片 | 法術 | 3 費 | 次元壁穩定度 +25。抽 1 張牌。若穩定度已為 100，改為對敵方全體造成 15 傷害。`

**v3.3 調整**：

- **效果不變**：穩定度 +25 / 抽 1 張 / 滿值改為全體 15 傷害
- **新增說明文字**：「此卡無法關閉已開啟的裂縫。」（與 S08 同步措辭）
- **Flavor 不變**

> N05 在「裂縫尚未開啟」階段仍是極強的中立傳說，但其價值在裂縫已開後純粹變為資源（穩定度數字本身不再影響裂縫狀態，但仍影響 v3.2 B.6 階段效果如治療減半等）。

### 6.4 A10 星落之劍

**原條目**：`A10 星落之劍 | 行動 | 7 費 | ★ | 對 1 個目標造成 ATK×2+20 傷害，無視 DEF 與守護。次元壁 ≤ 50 時額外 +15。`

**v3.3 調整**：

- **效果不變**
- **新增 flavor 補充**：A10 的「穩定度 ≤ 50 額外 +15」條件**與裂縫共生**——裂縫開啟（穩定度 < 50）的同時，A10 也獲得額外傷害。這是天然協同設計，無需修改數值。
- **Flavor 補充**：「星落不是天象，是次元壁第一次落淚那天，天空畫上的傷口。」

> A10 在裂縫機制上線後**戰略價值反而提升**——它是混沌派路線的「終結技」，能無視守護斬殺敵方滲透體或佔據敵方裂縫的兵力。

## 7. 模式相容性

### 7.1 內戰模式（PvP）

- **裂縫機制完全停用**
- 與 v3.2「內戰模式停用穩定度」一致——既然穩定度不存在，裂縫的觸發條件（穩定度 < 50）自然永遠不成立
- M01–M06 滲透體**不可加入任何玩家牌庫**，因此 PvP 也不會看到滲透體
- S15 / S16：可加入牌庫但無法在 PvP 中施放（施放條件「裂縫存在」永遠不成立）；建議 UI 將此類卡標記為「PvP 中無效」灰底
- F08（重作後）：同樣需要裂縫存在，PvP 中無法施放

### 7.2 試煉塔模式

- **全層級啟用裂縫機制**，不做層數特化調整
- 滲透體選擇邏輯（§3.2）的「當前回合數」對所有層級一致——不依層數調整池
- 1–30 層的 Boss 戰與巢穴交替戰皆套用此機制
- **預期影響**：低層（1–5 層）的新手戰鬥若穩定度跌破 50，會看到首次接觸 M01–M02；高層（25–30 層）累積戰術後可挑戰更刺激的 M06 級拉鋸戰
- 數值平衡留到 Stage 2 實作後依實戰資料調整，本 Stage 不為試煉塔做數值差異化

### 7.3 巢穴戰

- **啟用裂縫機制**
- 注意：v3.2 中部分巢穴本身會持續降穩定度（如「暗影門戶」-3/回合），這代表**巢穴戰中裂縫更容易也更早開啟**
- 這是**特性而非 bug**——巢穴本來就是「對抗黑暗次元擴張」的劇情戰場，裂縫提早出現符合敘事
- 滲透體選擇邏輯不變，但因回合進行較快觸發裂縫，玩家在巢穴戰中可能更早遇到 M03–M04 級單位
- 巢穴本身不會被滲透體取代或干擾——巢穴與裂縫位**完全獨立**（巢穴佔據敵方位置，裂縫位在戰場中央）

## 8. Flavor 與世界觀整合

### 8.1 核心敘事連結

依 worldbuilding：

| 世界觀元素 | 機制對應 |
|------------|----------|
| 「次元壁在眾神之戰後表面癒合，內部從未穩定」 | 裂縫不可逆——一旦撕開無法修補 |
| 「黑暗次元已開始滲透，次元壁裂縫正在擴大」 | 整個機制即「滲透初期」的戰術化具象 |
| 「裂痕日」習俗「不讓裂痕過夜」 | S08 次元修補的 flavor 重寫直接引用此習俗 |
| 主月碎裂、副月新生 | A10 星落之劍的 flavor 連結「次元壁第一次落淚」意象 |
| 魔族分類：惡魔類 / 原生類 / 魔化類 | M01–M06 嚴格從惡魔類 + 原生類抽取，**不使用魔化類**（魔化類為「腐人類」「腐獸類」等低階砲灰，不符合「滲透體」應有的重量感） |

### 8.2 組織連結（後續劇情鉤點）

以下為 Stage 2/3 / 後續資料片可發展的劇情鉤點，本 Stage 不實作但記錄：

- **空騎團**（中立、跨界偵查）：可有相關特殊裝備卡「空騎雷達」——預警裂縫即將開啟（穩定度跌至 55–60 區間時提示玩家）
- **聖殿騎士團**（反魔族使命）：對 M01–M06 滲透體有特化效果的兵力（如「破魔判官」對魔族卡傷害 +30%）
- **山獵人**（主角、身分神秘）：與次元滲透的關係未明，未來可作敘事核心。可預留一張「山獵人之箭」傳說行動，與裂縫互動方式待定。

### 8.3 Flavor text 一致性檢核

- 所有新卡（M01–M06、S15、S16）的 flavor text 已避免使用 v3.2 已存在的具體角色名（除非該角色已在 worldbuilding canon 中明確）
- M04 夢魔影刺 flavor「八條觸手者，曾在七位帝王的夢中漫步」——「七位帝王」是新隱喻，未綁定具體史實，留給後續創作填空
- S08 修改後 flavor 引用「裂痕日修碗」習俗，與 worldbuilding 節日描述完全對齊
- F08 重作 flavor「裂縫從來不是裂縫」呼應魔族 lore：「黑暗次元深處存在少數有意識的強大存在，動機不明」

## 9. 平衡與未來工作

### 9.1 平衡原則

依 brief §8「數值保守」：

- M01–M06 數值已設定為**偏中等**——M03 古魔（3 費 15/8/2）對比 H03 帝國弩砲（4 費 12/9/2）僅略強，留出 Stage 2 微調空間
- S15「免費部署」威力大，已限 3 次/場 + Open 條件鎖
- S16 抽 2 + 鬥志 +20 是強牌，已限 1 次/場 + 裂縫存在條件鎖
- F08 重作「滲透體池往上一階」是**雙刃懲罰機制**，刻意防止「F08 完全只對玩家有利」的不平衡

### 9.2 Stage 2 待驗證項目

實作完成後需在實戰中驗證的關鍵指標：

| 指標 | 期望範圍 | 驗證方法 |
|------|----------|----------|
| 裂縫平均開啟回合 | 試煉塔 6–10 / 巢穴 4–7 / Boss 戰 8–12 | Stage 2 自動化模擬 |
| 「玩家先佔據」vs「敵方先滲透」比例 | 約 40% / 60%（混沌略強勢） | 玩家行為日誌 |
| M06 巨魔出現後對戰局影響 | 玩家擊殺率 < 50% / 容忍 1–2 回合 | 平衡測試 |
| S15 使用率 | 30%–60% 牌局至少使用 1 次 | 玩家行為日誌 |

### 9.3 與其他系統的後續相容性備註

以下為 brief §9 列出的「先不處理但需保留設計空間」項目：

- **#4 英雄覺醒態**：未來覺醒態可能與裂縫互動（例如某英雄覺醒後可直接佔據敵方裂縫），現在不實作但保留 `rift.holder` 欄位的擴展性
- **#5 種族量表轉化**：佔據裂縫的 buff 目前僅 ATK×2 + DEF+5；未來可考慮「佔據時量表充能加速」作為某些種族專屬效果，但須在量表轉化系統定案後另設計
- **#8 三線戰場**：目前裂縫為**中央獨立欄位**；若未來改三線戰場，可重新定位為「中線專屬」或「跨線增益」，本 Stage 設計不影響該方向

### 9.4 已知設計風險

- **節奏緊迫風險**：開啟當回合震動、下回合立即滲透的設計，可能在玩家「不小心跌破 50」時造成措手不及。緩解：Stage 3 UI 必須有醒目的穩定度警示（如 55–60 紅色閃爍）
- **S15 連發風險**：3 次/場 × 免費部署高費兵力，理論上可在 3 回合內把全部 6 費以上兵力都打進裂縫位。緩解：留待 Stage 2 用實戰資料判斷是否需要降至 2 次或加冷卻
- **F08 雙刃失衡風險**：若敵方滲透強度提升過快而玩家佔據強化不夠，會變成「F08 完全是給敵方禮物」。緩解：〔穿透〕buff 是針對性破甲，對 M05 冰魔（3 DEF）和 M06 巨魔（7 DEF）效益最高，與「敵方升階池」的高 DEF 趨勢正好對沖
