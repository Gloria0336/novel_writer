# 星落戰紀 — 卡牌遊戲設計文件 v1.0

> 基於 novel_02 世界觀的單人 PvE 卡牌遊戲
> 類型：單英雄 + 支援牌組 | 節奏：15–25 分鐘/場 | 核心主題：種族對抗

---

## A. 遊戲概述

### A.1 核心體驗

玩家選擇一名來自七大種族的英雄，搭配自組的支援牌組，對抗由 AI 操控的魔族敵人。每場對局聚焦於單英雄的深度操作——英雄是場上唯一的戰鬥單位，牌組提供攻擊、防禦、增益與戰術支援。

### A.2 設計支柱

1. **種族身分即策略身分**：每個種族不只是換皮，而是從資源節奏、卡牌互動方式到勝利路線都有本質差異。
2. **英雄主導，牌組輔助**：英雄不是被動的血條，而是有自己的行動、被動技能與終極技的獨立戰場單位。牌組是英雄的「後勤團隊」。
3. **魔族不是砲灰**：每種敵人都有基於世界觀的獨特行為模式，打不同的魔族需要不同的策略。
4. **一場 15–25 分鐘**：節奏緊湊，前期建立、中期抉擇、後期爆發，不拖泥帶水。

### A.3 勝負條件

- **勝利**：將敵方英雄/Boss 的 HP 降至 0。
- **失敗**：我方英雄 HP 降至 0，或特定 Boss 的特殊敗北條件觸發（如次元壁崩潰倒數歸零）。

## B. 核心機制

### B.1 回合結構

每回合分為五個階段，雙方交替行動（玩家先攻）：

```
┌─────────────────────────────────────────────────┐
│  1. 補充階段（Draw Phase）                        │
│     → 抽 2 張牌（種族被動可能修改抽牌數）           │
│     → 魔力水晶 +1（上限 10）                      │
├─────────────────────────────────────────────────┤
│  2. 備戰階段（Preparation Phase）                 │
│     → 打出支援卡（消耗魔力）                       │
│     → 裝備卡立即附加到英雄身上                      │
│     → 持續型卡牌進入「支援區」                      │
├─────────────────────────────────────────────────┤
│  3. 行動階段（Action Phase）                      │
│     → 英雄執行 1 次行動（普攻/技能/終極技）          │
│     → 支援區的持續效果自動觸發                      │
├─────────────────────────────────────────────────┤
│  4. 敵方回應階段（Enemy Response Phase）           │
│     → AI 根據行為模式執行攻擊或特殊行動              │
│     → 玩家可打出「即時卡」回應（不消耗行動）          │
├─────────────────────────────────────────────────┤
│  5. 結算階段（Resolution Phase）                  │
│     → 結算所有延遲效果                             │
│     → 更新持續卡回合數                             │
│     → 檢查勝負條件                                │
└─────────────────────────────────────────────────┘
```

### B.2 資源系統

| 資源 | 說明 |
|------|------|
| **魔力水晶** | 每回合自動 +1，上限 10。用於打出支援卡。 |
| **鬥志（Morale）** | 英雄專屬資源，通過攻擊/受傷/特定卡牌累積，滿值時可施放終極技。初始 0，上限 100。 |
| **種族量表** | 每個種族有獨特的第三資源，詳見 C 節。 |

### B.4 敵方行為預告系統

AI 敵人不是隨機行動。每回合結束時，敵方會翻開 1–3 張「意圖卡」，預告下回合將做什麼（類似 Slay the Spire 的意圖系統，但更複雜）：

- **紅色意圖**：攻擊（顯示傷害數值）
- **藍色意圖**：防禦/強化自身
- **紫色意圖**：特殊技能（如夢魔的精神控制、咒魔的詛咒）
- **黑色意圖**：次元滲透（累積腐化值，見 B.5）
- **金色意圖**：召喚增援

部分高階 Boss 會出現「模糊意圖」（以 ？ 顯示），需要特定種族技能或偵查卡才能揭示。

### B.5 腐化機制（全局張力）

每場對局有一個共享的 **次元壁穩定度** 量表，初始 100。

- 魔族的黑色意圖會降低穩定度
- 部分強力魔族卡牌也會降低穩定度
- 穩定度每降低 25 點，觸發一個環境負面效果：
  - **75**：腐植滋生——支援區上限 -1
  - **50**：裂縫擴大——敵方每回合額外行動一次
  - **25**：黑暗湧入——所有治療效果減半
  - **0** ：次元崩潰——玩家直接敗北
- 部分卡牌或英雄技能可以恢復穩定度

這個機制讓每場對局都有隱性的時間壓力，避免無限拖延的龜縮策略。

## C. 英雄系統

### C.1 英雄通用屬性

| 屬性 | 說明 |
|------|------|
| HP | 生命值，歸零即敗。依種族不同，範圍 60–120。 |
| ATK | 基礎攻擊力，普攻傷害 = ATK + 卡牌加成。 |
| DEF | 防禦力，受到傷害時固定減免。 |
| SPD | 速度，決定每回合可使用的即時卡數量（1–3）。 |

每位英雄擁有：
- **1 個種族被動**（永久生效）
- **2 個主動技能**（消耗鬥志）
- **1 個終極技**（鬥志滿 100 時可施放，施放後清零）
- **3 個裝備欄位**（武器 / 防具 / 飾品各一）

### C.2 六大可選種族英雄

---

#### 【人類】武鬥騎士

> *三族聯盟的劍與盾。適應力最強，節奏最穩。*

| 屬性 | 數值 |
|------|------|
| HP | 90 |
| ATK | 12 |
| DEF | 6 |
| SPD | 2 |

**種族量表：鬥氣（Aura）**
- 每次普攻或受到傷害時 +10 鬥氣
- 鬥氣可消耗來強化下一次攻擊（每 10 點鬥氣 = +3 ATK）
- 鬥氣滿 100 時進入「鬥氣外放」狀態，持續 2 回合內所有攻擊附帶穿透（無視 DEF）

**被動技能｜軍團適應**
每場對局的第 3、6、9 回合，永久獲得 +1 ATK 或 +1 DEF（玩家選擇）。模擬人類的快速學習與成長。

**主動技能 A｜盾擊**（消耗 30 鬥志）
造成 ATK×1.5 傷害，並獲得等量護甲。攻防一體的核心技。

**主動技能 B｜戰吼**（消耗 40 鬥志）
下 2 回合抽牌 +1，且所有攻擊卡費用 -1。節奏加速技。

**終極技｜平民之怒**（消耗 100 鬥志）
造成 ATK×3 傷害。若本場已裝備 2 件以上裝備，額外造成 15 點固定傷害。

**設計意圖**：人類是最「正統」的英雄選擇，節奏平穩，容錯率高，適合新手。鬥氣系統獎勵積極交戰而非被動防守。

---

#### 【精靈】魔導賢者

> *萬年知識的活字典。魔力充沛，連鎖施法無人能及。*

| 屬性 | 數值 |
|------|------|
| HP | 70 |
| ATK | 8 |
| DEF | 4 |
| SPD | 3 |

**種族量表：共鳴（Resonance）**
- 每打出一張法術卡，共鳴 +1（每回合重置）
- 同一回合內共鳴越高，後續法術卡效果越強（每層共鳴 = 法術效果 +15%）
- 單回合共鳴達到 5 層時觸發「詠唱完成」：下一張法術卡費用為 0

**被動技能｜魔力親和**
每回合魔力水晶額外 +1（實際上限 11）。精靈永遠比其他種族多 1 點魔力。

**主動技能 A｜星辰鎖鏈**（消耗 30 鬥志）
對敵方施加「減速」2 回合（敵方意圖卡數量 -1）。控制向技能。

**主動技能 B｜知識灌注**（消耗 40 鬥志）
從棄牌堆回收 2 張法術卡到手牌。資源回收技。

**終極技｜古語終章**（消耗 100 鬥志）
造成 10 + (本場已使用法術卡數量 ×3) 的魔法傷害。越到後期威力越恐怖。

**設計意圖**：精靈獎勵「一回合內連鎖多張法術」的操作風格，上限極高但需要手牌管理能力。SPD 3 讓精靈有最多的即時卡回應空間。

---

#### 【矮人】鍛造大師

> *他們打造的不只是武器，是贏得戰爭的方式。*

| 屬性 | 數值 |
|------|------|
| HP | 110 |
| ATK | 10 |
| DEF | 8 |
| SPD | 1 |

**種族量表：爐火（Forge Heat）**
- 每次裝備一件裝備卡，爐火 +20
- 每次使用「鍛造」將手牌轉化為裝備時，爐火 +30
- 爐火滿 100 時進入「大師之爐」狀態：下一件裝備獲得雙倍數值

**被動技能｜堅韌鑄骨**
受到傷害時，若傷害值 ≤ DEF，完全免疫該次傷害（而非僅減免）。矮人對小傷害幾乎無感。

**主動技能 A｜鍛造**（消耗 20 鬥志）
將手牌中任意 1 張非裝備卡轉化為一件隨機裝備。裝備品質與被消耗卡牌的稀有度相關。

**主動技能 B｜合金強化**（消耗 40 鬥志）
選擇一件已裝備的裝備，永久 +3 數值。可疊加。

**終極技｜祖傳神器**（消耗 100 鬥志）
憑空創造一件「傳說裝備」並立即裝備。傳說裝備數值為普通裝備的 3 倍，且附帶獨特效果（從預設池中隨機選取）。若裝備欄已滿，玩家選擇替換一件。

**設計意圖**：矮人的策略核心是「改造手牌為裝備」，慢慢堆疊出無法摧毀的裝備牆。SPD 1 意味著幾乎沒有即時卡回應空間，需要靠高 DEF 和裝備硬扛。

#### 【妖族】幻形師

> *萬物皆可為形，萬形皆可為我。*

| 屬性 | 數值 |
|------|------|
| HP | 80 |
| ATK | 11 |
| DEF | 5 |
| SPD | 2 |

**種族量表：靈蘊（Essence）**
- 每回合自動 +10 靈蘊
- 靈蘊是變身的燃料——維持非人形態每回合消耗靈蘊
- 回到人形時，靈蘊停止消耗但也不再自然增長（需靠卡牌補充）

**被動技能｜雙形切換**
妖族英雄擁有兩種形態，每回合開始時可免費切換一次：
- **人形**：使用法術卡效果 +20%，但普攻傷害 -3
- **妖形**：普攻傷害 +5，且普攻附帶「撕裂」（每回合流血 3 點，可疊加），但無法使用法術卡

**主動技能 A｜靈蘊爆發**（消耗 30 鬥志）
立即獲得 50 靈蘊。若在妖形下使用，額外獲得 1 次普攻機會。

**主動技能 B｜鏡像幻術**（消耗 40 鬥志）
生成一個幻影，吸收下一次敵方攻擊傷害。若在人形下使用，幻影存在 2 回合而非 1 回合。

**終極技｜祖靈降臨**（消耗 100 鬥志）
進入「始祖妖形」3 回合——同時享有人形與妖形的所有加成，且靈蘊消耗為 0。

**設計意圖**：妖族的核心體驗是在兩種形態間切換，根據敵方行動選擇最佳形態。操作天花板高，需要預判敵方意圖。

#### 【獸族】蠻血戰士

> *文明是別人的事。活下來才是我的事。*

| 屬性 | 數值 |
|------|------|
| HP | 100 |
| ATK | 14 |
| DEF | 3 |
| SPD | 2 |

**種族量表：血怒（Blood Rage）**
- 每損失 10% HP，血怒 +1 層（最多 10 層）
- 每層血怒 = +2 ATK
- 血怒 5 層以上時解鎖「嗜血」：普攻附帶吸血（恢復造成傷害的 20%）
- 治療會降低血怒層數（每恢復 10% HP，-1 層）

**被動技能｜野性直覺**
敵方的紅色意圖（攻擊）傷害對獸族英雄減少 20%。獸族天生擅長在肉搏中存活。

**主動技能 A｜撕咬**（消耗 20 鬥志）
造成 ATK×1.2 傷害。若血怒 ≥ 3 層，改為 ATK×1.8。低門檻高頻率的輸出技。

**主動技能 B｜戰嚎**（消耗 50 鬥志）
自身受到 15 點傷害（無法被減免），但獲得 5 層血怒。暴力加速的雙刃劍。

**終極技｜祖獸覺醒**（消耗 100 鬥志）
造成 (ATK + 血怒層數×5) ×2 的傷害。在 10 層血怒下使用，這是全遊戲單次最高傷害。施放後血怒清零、HP 恢復 30%。

**設計意圖**：獸族是純粹的攻擊種族，核心矛盾在於「受傷越多越強，但死了就沒了」。需要精確控制自己的血線，在刀尖上跳舞。

#### 【半神族】遺落神裔

> *他們不知道自己是什麼。但所有人都知道他們不該存在。*

| 屬性 | 數值 |
|------|------|
| HP | 60 |
| ATK | 10 |
| DEF | 4 |
| SPD | 2 |

**種族量表：神力殘響（Divine Echo）**
- 不自然增長，只能通過特定卡牌或條件獲得
- 是使用神力技能的唯一燃料（不消耗魔力水晶）
- 每次使用神力技能後，獲得 1 層「透支」debuff（每層 = 受到傷害 +5%）

**被動技能｜不朽殘片**
每場對局第一次 HP 歸零時，不會死亡，而是以 1 HP 存活並清除所有 debuff。僅觸發一次。

**主動技能 A｜神言：修復**（消耗 20 神力殘響）
恢復 25 HP 並移除 1 層透支。半神族唯一的自我修復手段。

**主動技能 B｜神言：斷裂**（消耗 30 神力殘響）
對敵方造成 30 點固定傷害（無視護甲與防禦）。代價是獲得 2 層透支。

**終極技｜神域片刻**（消耗 100 鬥志 + 50 神力殘響）
接下來 2 回合內，英雄免疫所有傷害，且所有攻擊造成雙倍傷害。結束後獲得 5 層透支。

**設計意圖**：半神族是高風險高回報的種族。神力極強但有嚴重副作用，60 HP 的血量是全種族最低。核心抉擇在於「何時使用神力、使用多少」——過度使用會讓透支疊加到致命程度，不使用又打不出足夠傷害。不朽殘片是最後的安全網。

### C.3 英雄解鎖與成長

- **初始解鎖**：人類、精靈、矮人（三族聯盟核心成員）
- **通關解鎖**：以三族任一英雄通關故事模式後，依序解鎖妖族 → 獸族 → 半神族
- **每位英雄有獨立的經驗值**：通關對局獲得經驗值，升級解鎖新的起始卡牌與外觀

## D. 卡牌系統

### D.1 卡牌分類

所有卡牌分為 **六大類別**，每類有獨特的邊框顏色與視覺標識：

| 類別 | 邊框色 | 功能 | 規則 |
|------|--------|------|------|
| **攻擊卡** | 紅 | 對敵方造成傷害 | 在備戰階段打出，行動階段結算 |
| **防禦卡** | 藍 | 獲得護甲或減傷效果 | 備戰階段打出，當回合生效 |
| **法術卡** | 紫 | 特殊效果（控制、debuff、回復、操作場地等） | 備戰階段打出，效果多樣 |
| **裝備卡** | 橙 | 附加到英雄裝備欄，提供持續增益 | 打出後永久生效，直到被替換或摧毀 |
| **即時卡** | 綠 | 可在敵方回應階段打出的反應牌 | 不消耗行動，消耗 SPD 次數 |
| **種族卡** | 金 | 限定特定種族使用的強力卡牌 | 只有對應種族英雄才能放入牌組 |

### D.2 卡牌稀有度

| 稀有度 | 標記 | 牌組限制 |
|--------|------|----------|
| 普通 | ● | 每張最多 3 份 |
| 精良 | ●● | 每張最多 2 份 |
| 稀有 | ●●● | 每張最多 1 份 |
| 傳說 | ★ | 每張最多 1 份，全牌組最多 3 張傳說 |

### D.3 牌組構築規則

- 牌組大小：**30 張**（固定）

## G. 數值平衡框架

### G.1 傷害經濟

基準假設：一場對局 10–15 回合結束。

| 指標 | 數值範圍 |
|------|----------|
| 英雄每回合平均輸出 | 15–25 傷害 |
| 敵方 Boss 總 HP | 150–250 |
| 敵方每回合平均輸出 | 10–18 傷害 |
| 英雄總 HP | 60–120 |
| 每回合平均可用魔力 | 3–7（依回合數遞增） |

### G.2 平衡公式

- 英雄預期存活回合 = 英雄 HP ÷ (敵方平均輸出 - 每回合平均減傷) ≈ 12–18 回合
- 敵方預期存活回合 = 敵方 HP ÷ 英雄平均輸出 ≈ 8–14 回合
- 差值即為玩家的策略空間——需要通過卡牌操作將存活回合拉長或將擊殺回合縮短
