# 星落戰紀 — 卡牌遊戲設計文件 v3.3

> 基於 novel_02 世界觀的單人 PvE 種族戰爭卡牌遊戲
> 類型：種族×職業框架 + 個體化英雄 + 兵力戰場 | 節奏：15–25 分鐘/場
> 戰鬥架構：闇影詩章式自由回合制
>
> **v3.3 主要新增**：次元滲透裂縫（Rift Slot）機制——把「次元壁穩定度」從單向失敗計時器升級為雙刃戰術資源。詳細規格見 `rift_slot_design_v1.md`。

---

## A. 遊戲概述

### A.1 核心體驗

玩家選擇一名英雄投入種族戰爭。每位英雄由 **種族框架 + 職業框架 + 個體特質** 三層定義——種族決定軍團特性與資源體系，職業決定戰場機制與數值傾向，而英雄個體則擁有完全獨立的技能組與數值微調。

同樣是人類·指揮官，不同角色的戰鬥風格可以截然不同——英雄卡直接對應小說中的角色，每位角色都有專屬的技能設計。

### A.2 設計支柱

1. **角色即英雄**：英雄卡直接對應小說角色，技能反映角色特質。種族和職業提供共通框架，但每個角色的技能都是獨一無二的。
2. **無限擴展**：英雄數量不受種族×職業矩陣限制。同一種族同一職業可以有多位英雄，隨小說角色逐步推出。
3. **戰場是核心**：兵力牌在戰場中線交鋒，英雄指揮並適時介入。
4. **雙軌攻擊規則**：兵力遵循「兵力優先」；英雄行動遵循「守護優先」。
5. **次元壁雙刃**（v3.3 新增）：穩定度不再只是失敗計時器，跨過 50 線後會開啟「次元滲透裂縫」雙刃資源，玩家需在守序與混沌間做戰略選擇。

### A.3 勝負條件

- **Boss 戰**：將敵方英雄 HP 降至 0。
- **巢穴戰**：將敵方巢穴 HP 降至 0。
- **失敗**：我方英雄 HP 降至 0，或次元壁穩定度歸零。

## B. 核心機制

### B.1 回合結構

```
❶ 回合開始（自動）
   → 抽 1 張（手牌上限 9）→ 魔力上限 +1（最高 10）並全回復 → 觸發回合開始效果

❷ 主要階段（自由操作）
   以下動作可任意順序、任意次數：
   部署兵力 / 施放法術 / 使用行動卡 / 裝備 / 放置場地 / 指揮攻擊 / 英雄技能

❸ 回合結束 → 結算觸發效果 → 切換敵方回合
   v3.3：若有開啟的次元滲透裂縫，結算「裂縫震動倒數」（見 §B.6.2）

❹ 敵方回合 → AI 執行相同流程。玩家無法操作。
```

起始：各抽 3 張可換牌。第 N 回合魔力上限 = N。玩家固定先攻。

### B.2 攻擊規則（雙軌制）

| | 兵力牌攻擊 | 英雄行動卡/技能 |
|------|-----------|---------------|
| 限制 | **兵力優先**：有敵方兵力時只能打兵力 | **守護優先**：可繞過普通兵力，但必須先打守護 |
| 無敵方兵力時 | 直接打臉 | 直接打臉 |
| 傷害模型 | 雙方交換傷害 | 單向傷害 |
| 暈眩 | 部署當回合不可攻擊 | 無暈眩 |

### B.3 場地佈局

每方最多 5 個兵力欄位。兵力傷害永久，HP 歸零即摧毀。場地卡同一時間 1 張。

**v3.3 新增——次元滲透裂縫位**：當次元壁穩定度跌破 50 時，戰場中央會出現一個「裂縫位」獨立欄位（不佔敵我兵力欄）。場上至多 1 個裂縫，與場地卡獨立共存。詳見 §B.6.2 與 `rift_slot_design_v1.md`。

### B.4 資源系統

| 資源 | 說明 |
|------|------|
| **魔力水晶** | 第 N 回合上限 = N（最高 10，精靈族 13）。每回合全回復。 |
| **鬥志** | 擊殺兵力 +15 / 行動卡命中 +10 / 我方兵力被殺 +5 / **擊殺敵方滲透體 +10**（v3.3）。滿 100 施放終極技。 |
| **種族量表** | 由種族定義框架名稱與基礎機制，英雄個體可自訂觸發條件與效果。 |

### B.5 關鍵字系統

| 關鍵字 | 效果 |
|--------|------|
| 守護 | 敵方英雄行動卡/技能必須優先攻擊守護單位 |
| 突進 | 部署當回合可攻擊敵方兵力 |
| 疾走 | 部署當回合可攻擊任何目標 |
| 必殺 | 交戰敵方兵力必定摧毀 |
| 汲取 | 攻擊造傷時英雄回復等量 HP |
| 威壓 | 不可被敵方兵力選為攻擊目標 |
| 穿透 | 無視目標 DEF |
| 入場曲 | 從手牌部署時觸發 |
| 謝幕曲 | 被摧毀時觸發 |

### B.6 腐化機制 / 次元滲透

#### B.6.1 次元壁穩定度（v3.2 保留）

次元壁穩定度初始 100。階段效果：

| 穩定度 | 效果 |
|--------|------|
| ≤ 75 | 兵力欄 -1 |
| ≤ 50 | 敵方額外部署 +「**次元滲透裂縫**」開啟（v3.3） |
| ≤ 25 | 治療減半 |
| 0 | 直接敗北 |

**內戰模式停用**整個腐化機制（含裂縫）。

#### B.6.2 次元滲透裂縫（v3.3 新增）

當穩定度**首次**跌破 50 時，戰場中央出現「次元滲透裂縫」欄位。**規則摘要**：

- **不可逆**：裂縫一旦開啟即持續至戰鬥結束，穩定度回升至 ≥ 50 也不關閉
- **數量上限**：場上至多 1 個裂縫
- **三種狀態**：Open（空置）/ 玩家佔據 / 敵方佔據
- **滲透節奏**：開啟當回合震動告示，下一回合結束自動滲透 1 個敵方魔族單位（從 M01–M06 抽取）
- **佔據加乘**：佔據單位獲得 ATK×2、DEF+5（敵我對稱），HP 不變
- **拉鋸戰**：佔據者陣亡 → 裂縫回到 Open 狀態，倒數重設

> 完整規格、滲透體白名單、新卡（S15, S16）、F08 重作等內容詳見 **`rift_slot_design_v1.md`**。

### B.7 敵方意圖預告（Boss 戰）

紅=攻擊 / 藍=部署 / 紫=法術 / 金=特殊 / ？=模糊

> **v3.3 異動**：原 v3.2 規劃的「黑=次元滲透」意圖**不再採用**。次元滲透改由裂縫本身狀態驅動（「裂縫震動」視覺告示），不擴張 AI 意圖系統。

## C. 英雄系統

### C.1 三層架構

每位英雄由三層獨立系統疊加而成：

```
┌─────────────────────────────────────────────────────────────┐
│ 第一層：種族框架                                              │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ • 種族量表名稱與基礎規則（如「軍令」「共鳴」「血怒」等）     │ │
│ │ • 種族數值修正（HP / ATK / DEF / CMD）                    │ │
│ │ • 牌組構築限制（各類別卡牌配額）                            │ │
│ │ • 種族專屬卡池（金邊框種族卡）                              │ │
│ │                                                          │ │
│ │ 同種族的所有英雄共享以上框架。                               │ │
│ └─────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│ 第二層：職業框架                                              │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ • 職業關鍵字（如「號令」「超載」「狂暴」等）                 │ │
│ │ • 職業數值修正（HP / ATK / DEF / CMD）                    │ │
│ │                                                          │ │
│ │ 同職業的所有英雄共享以上框架。                               │ │
│ │ 職業不影響牌組限制，不提供技能。                             │ │
│ └─────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│ 第三層：英雄個體                                              │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ • 個體數值微調（±1–5 的細部調整）                          │ │
│ │ • 被動技能 ×1–2（完全客製，呼應角色特質）                   │ │
│ │ • 主動技能 ×2–4（完全客製，消耗鬥志）                      │ │
│ │ • 終極技 ×1（完全客製，鬥志 100 施放）                     │ │
│ │ • 種族量表個人化（自訂觸發條件 / 效果 / 滿值獎勵）          │ │
│ │                                                          │ │
│ │ 每位英雄完全獨立。同種族同職業的不同英雄技能完全不同。        │ │
│ └─────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**數值計算公式：**

```
最終數值 = 基礎值 + 種族修正 + 職業修正 + 英雄個體微調

基礎值：HP 80 / ATK 10 / DEF 5 / CMD 5
```

### C.2 英雄稀有度

英雄卡本身有稀有度，影響技能欄位數量：

| 英雄稀有度 | 被動技能 | 主動技能 | 終極技 | 說明 |
|-----------|---------|---------|--------|------|
| R（稀有） | 1 | 2 | 1 | 基礎英雄，數值微調幅度較小（±1–2） |
| SR（超稀有） | 1–2 | 3 | 1 | 進階英雄，數值微調幅度中等（±1–3） |
| SSR（極稀有） | 2 | 3–4 | 1 | 頂級英雄，數值微調幅度較大（±1–5） |

> 技能欄位數量影響英雄的策略深度與上限，但不代表高稀有度英雄一定更強——數值平衡確保 R 級英雄在適當策略下也能通關。

## F. 遊戲模式

### F.1 試煉塔

30 層 Roguelike。每 5 層 Boss，其餘巢穴。勝利三選一獎勵。難度遞增。隨機事件。

**v3.3**：全層級啟用裂縫機制，不做層數差異化。滲透體選擇邏輯依當前回合數，所有層級一致。

### F.2 內戰模式

種族 vs 種族。AI 操控完整英雄。次元壁停用。天象正常。

**v3.3**：次元壁停用代表**裂縫機制完全停用**。S15 / S16 / F08（重作版）在 PvP 無法施放。M01–M06 不會出現。

### F.3 種族挑戰

限定種族 vs 剋星敵人。解鎖條件：該種族通過試煉塔第 15 層。

**v3.3**：啟用裂縫機制。

### F.4 巢穴戰

**v3.3**：啟用裂縫機制。部分巢穴本身降穩定度，會讓裂縫更早出現——這是設計特性。
