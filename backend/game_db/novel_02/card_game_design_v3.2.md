# 星落戰紀 — 卡牌遊戲設計文件 v3.2

> 基於 novel_02 世界觀的單人 PvE 種族戰爭卡牌遊戲
> 類型：種族×職業框架 + 個體化英雄 + 兵力戰場 | 節奏：15–25 分鐘/場
> 戰鬥架構：闇影詩章式自由回合制

---

## A. 遊戲概述

### A.1 核心體驗

玩家選擇一名英雄投入種族戰爭。每位英雄由 **種族框架 + 職業框架 + 個體特質** 三層定義——種族決定軍團特性與資源體系，職業決定戰場機制與數值傾向，而英雄個體則擁有完全獨立的技能組與數值微調。

同樣是人類·指揮官，不同角色的戰鬥風格可以截然不同——英雄卡直接對應小說中的角色，每位角色都有專屬的技能設計。

### A.2 設計支柱

1. **角色即英雄**：英雄卡直接對應小說角色，技能反映角色特質。種族和職業提供共通框架，但每個角色的技能都是獨一無二的。
2. **無限擴展**：英雄數量不受種族×職業矩陣限制。同一種族同一職業可以有多位英雄，隨小說角色逐步推出。
3. **戰場是核心**：兵力牌在戰場中線交鋒，英雄指揮並適時介入。
4. **雙軌攻擊規則**：兵力遵循「兵力優先」；英雄行動遵循「守護優先」。

### A.3 勝負條件

- **Boss 戰**：將敵方英雄 HP 降至 0。
- **巢穴戰**：將敵方巢穴 HP 降至 0。
- **失敗**：我方英雄 HP 降至 0，或次元壁穩定度歸零。

## B. 核心機制

### B.1 回合結構

```
❶ 回合開始（自動）
   → 抽 1 張（手牌上限 9）→ 魔力上限 +1（最高 10）並全回復 → 觸發回合開始效果

❷ 主要階段（自由操作）
   以下動作可任意順序、任意次數：
   部署兵力 / 施放法術 / 使用行動卡 / 裝備 / 放置場地 / 指揮攻擊 / 英雄技能

❸ 回合結束 → 結算觸發效果 → 切換敵方回合

❹ 敵方回合 → AI 執行相同流程。玩家無法操作。Boss 戰結束時顯示意圖預告。
```

起始：各抽 3 張可換牌。第 N 回合魔力上限 = N。玩家固定先攻。

### B.2 攻擊規則（雙軌制）

| | 兵力牌攻擊 | 英雄行動卡/技能 |
|------|-----------|---------------|
| 限制 | **兵力優先**：有敵方兵力時只能打兵力 | **守護優先**：可繞過普通兵力，但必須先打守護 |
| 無敵方兵力時 | 直接打臉 | 直接打臉 |
| 傷害模型 | 雙方交換傷害 | 單向傷害 |
| 暈眩 | 部署當回合不可攻擊 | 無暈眩 |

### B.3 場地佈局

每方最多 5 個兵力欄位。兵力傷害永久，HP 歸零即摧毀。場地卡同一時間 1 張。

### B.4 資源系統

| 資源 | 說明 |
|------|------|
| **魔力水晶** | 第 N 回合上限 = N（最高 10，精靈族 13）。每回合全回復。 |
| **鬥志** | 擊殺兵力 +15 / 行動卡命中 +10 / 我方兵力被殺 +5。滿 100 施放終極技。 |
| **種族量表** | 由種族定義框架名稱與基礎機制，英雄個體可自訂觸發條件與效果。 |

### B.5 關鍵字系統

| 關鍵字 | 效果 |
|--------|------|
| 守護 | 敵方英雄行動卡/技能必須優先攻擊守護單位 |
| 突進 | 部署當回合可攻擊敵方兵力 |
| 疾走 | 部署當回合可攻擊任何目標 |
| 必殺 | 交戰敵方兵力必定摧毀 |
| 汲取 | 攻擊造傷時英雄回復等量 HP |
| 威壓 | 不可被敵方兵力選為攻擊目標 |
| 穿透 | 無視目標 DEF |
| 入場曲 | 從手牌部署時觸發 |
| 謝幕曲 | 被摧毀時觸發 |

### B.6 腐化機制

次元壁穩定度初始 100。75 時兵力欄 -1 / 50 時敵方額外部署 / 25 時治療減半 / 0 時直接敗北。內戰模式停用。

### B.7 敵方意圖預告（Boss 戰）

紅=攻擊 / 藍=部署 / 紫=法術 / 黑=次元滲透 / 金=特殊 / ？=模糊

## C. 英雄系統

### C.1 三層架構

每位英雄由三層獨立系統疊加而成：

```
┌─────────────────────────────────────────────────────────────┐
│ 第一層：種族框架                                              │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ • 種族量表名稱與基礎規則（如「軍令」「共鳴」「血怒」等）     │ │
│ │ • 種族數值修正（HP / ATK / DEF / CMD）                    │ │
│ │ • 牌組構築限制（各類別卡牌配額）                            │ │
│ │ • 種族專屬卡池（金邊框種族卡）                              │ │
│ │                                                          │ │
│ │ 同種族的所有英雄共享以上框架。                               │ │
│ └─────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│ 第二層：職業框架                                              │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ • 職業關鍵字（如「號令」「超載」「狂暴」等）                 │ │
│ │ • 職業數值修正（HP / ATK / DEF / CMD）                    │ │
│ │                                                          │ │
│ │ 同職業的所有英雄共享以上框架。                               │ │
│ │ 職業不影響牌組限制，不提供技能。                             │ │
│ └─────────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│ 第三層：英雄個體                                              │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ • 個體數值微調（±1–5 的細部調整）                          │ │
│ │ • 被動技能 ×1–2（完全客製，呼應角色特質）                   │ │
│ │ • 主動技能 ×2–4（完全客製，消耗鬥志）                      │ │
│ │ • 終極技 ×1（完全客製，鬥志 100 施放）                     │ │
│ │ • 種族量表個人化（自訂觸發條件 / 效果 / 滿值獎勵）          │ │
│ │                                                          │ │
│ │ 每位英雄完全獨立。同種族同職業的不同英雄技能完全不同。        │ │
│ └─────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**數值計算公式：**

```
最終數值 = 基礎值 + 種族修正 + 職業修正 + 英雄個體微調

基礎值：HP 80 / ATK 10 / DEF 5 / CMD 5
```

### C.2 英雄稀有度

英雄卡本身有稀有度，影響技能欄位數量：

| 英雄稀有度 | 被動技能 | 主動技能 | 終極技 | 說明 |
|-----------|---------|---------|--------|------|
| R（稀有） | 1 | 2 | 1 | 基礎英雄，數值微調幅度較小（±1–2） |
| SR（超稀有） | 1–2 | 3 | 1 | 進階英雄，數值微調幅度中等（±1–3） |
| SSR（極稀有） | 2 | 3–4 | 1 | 頂級英雄，數值微調幅度較大（±1–5） |

> 技能欄位數量影響英雄的策略深度與上限，但不代表高稀有度英雄一定更強——數值平衡確保 R 級英雄在適當策略下也能通關。

### C.3 六大種族框架

每個種族定義「你帶領什麼樣的軍隊」。

#### 人類

|------------|-------|--------|--------|--------|

**種族量表框架：軍令（Command Order）**
- 基礎規則：與兵力部署和兵力存活相關的資源
- 英雄個體可自訂：累積方式（部署時 / 攻擊時 / 兵力存活時）、消耗效果（強化兵力 / 召喚兵力 / 群體增益）、滿值觸發

**種族共通特色：**
- 兵力牌在戰場上會獲得某種形式的增益（具體增益由英雄被動決定）
- 牌組兵力卡配額最多

**牌組限制：** 兵力 14–18 / 行動 4–6 / 法術 2–4 / 裝備 2–3 / 場地 1–2

**種族共通特色：**
- 魔力水晶上限 13（而非 10）
- 法術體系有某種形式的連鎖/累積加成

### C.4 六大職業框架（可擴展）

每個職業定義「你在戰場上扮演什麼角色」。職業提供的是被動機制框架（職業關鍵字），不提供技能——技能全由英雄個體決定。

---

#### 指揮官

| 職業數值修正 | HP +0 | ATK -2 | DEF +0 | CMD +2 |
|------------|-------|--------|--------|--------|

**職業關鍵字：號令**
指揮官的效果會隨場上己方兵力數量增強。同一回合內場上兵力越多，指揮官的行動卡傷害和技能效果越強。

> 加成公式（基礎）：場上每有 1 個己方兵力，行動卡傷害 +5%。英雄個體可調整此比例或新增額外觸發。

**定位：** 兵力導向的後方指揮型。CMD 疊加讓指揮官的兵力欄位和兵力增益最高。

---

#### 法師

| 職業數值修正 | HP -10 | ATK +0 | DEF -1 | CMD +0 |
|------------|--------|--------|--------|--------|

**職業關鍵字：超載**
法師每回合使用第 3 張法術卡起，每多使用 1 張法術卡額外獲得 1 點臨時魔力（本回合有效）。鼓勵法術連鎖。

> 英雄個體可調整超載的觸發門檻、額外效果（如超載時法術傷害 +X%）。

**定位：** 法術導向的輸出/控制型。低生存力換取法術連鎖爆發。

---

#### 鍛造師

| 職業數值修正 | HP +10 | ATK +0 | DEF +2 | CMD +0 |
|------------|--------|--------|--------|--------|

**職業關鍵字：改造**
鍛造師可以在主要階段中消耗 1 點魔力將手牌中 1 張非裝備卡轉化為隨機裝備卡（每回合限 1 次，不消耗鬥志）。

> 英雄個體可調整改造的附加效果（如改造時爐火 +X、改造產物的品質規則）。

**定位：** 裝備/器具導向的成長型。高耐久，靠裝備疊加滾雪球。

---

#### 幻術師

| 職業數值修正 | HP +0 | ATK +1 | DEF +0 | CMD +0 |
|------------|-------|--------|--------|--------|

**職業關鍵字：幻影**
幻術師可以生成「幻影兵力」（HP 1 / ATK 0 / DEF 0 / 守護）。幻影不消耗手牌，直接出現在戰場上。每回合可免費生成 1 個幻影（佔用兵力欄位）。

> 英雄個體可調整幻影的屬性、附加效果（如謝幕曲造傷、幻影存活時的光環效果）。

**定位：** 欺騙/干擾導向的靈活型。用幻影守護消耗敵方資源，配合其他手段輸出。

---

#### 狂戰士

| 職業數值修正 | HP +0 | ATK +3 | DEF -2 | CMD +0 |
|------------|-------|--------|--------|--------|

**職業關鍵字：狂暴**
狂戰士的行動卡傷害隨英雄已損失 HP 百分比增加。每損失 10% 最大 HP = 行動卡最終傷害 +8%。

> 英雄個體可調整狂暴的加成比例、附加效果（如殘血時解鎖額外行動、吸血等）。

**定位：** 個體戰鬥導向的激進型。高 ATK 低 DEF，越殘血越致命。

---

#### 神官

| 職業數值修正 | HP +5 | ATK -2 | DEF +1 | CMD +0 |
|------------|-------|--------|--------|--------|

**職業關鍵字：祝福**
神官使用恢復類效果時（包含法術卡、技能），恢復量額外 +50%。

> 英雄個體可調整祝福的觸發範圍（僅英雄 / 包含兵力 / 全體）和附加效果（如祝福時移除 debuff、增加護甲）。

**定位：** 續航/支援導向的持久型。拉長對局，靠持續回復消耗敵方。

### C.5 英雄卡範例（Demo 英雄）

以下 6 位英雄為 Demo 版初始英雄，展示三層系統如何組合。

---

#### Demo 英雄 #1：軍團統帥

| 欄位 | 內容 |
|------|------|
| 種族 | 人類 |
| 職業 | 指揮官 |
| 稀有度 | SR |
| 個體微調 | HP +0 / ATK +0 / DEF +0 / CMD +0 |

| 最終數值 | HP 80 | ATK 6 | DEF 5 | CMD 9 |
|---------|-------|-------|-------|-------|
| 來源 | 80+0+0+0 | 10-2-2+0 | 5+0+0+0 | 5+2+2+0 |

**種族量表「軍令」個人化：**
- 累積：部署兵力 +10 / 場上每存活 1 兵力每回合 +5
- 消耗：消耗 50 軍令可讓所有兵力本回合 ATK +3

**被動技能｜軍團旗幟**
所有我方兵力 +2 DEF、+5 HP。場上每多 1 個兵力額外 +1 DEF、+2 HP（最多 5 層）。

**主動技能 A｜徵召令**（消耗 30 鬥志）
從牌庫搜尋 1 張兵力牌加入手牌，部署費用 -2。

**主動技能 B｜堅守陣線**（消耗 40 鬥志）
所有兵力本回合 DEF ×2，無法被一擊殺（最低 1 HP）。

**主動技能 C｜戰場調度**（消耗 20 鬥志）

### C.6 未來英雄設計指南

新英雄的設計流程：

```
1. 確認角色身分 → 從小說中選取角色
2. 指定種族 → 對應角色的種族（或保留調整空間）
3. 指定職業 → 對應角色的戰鬥風格
4. 決定稀有度 → R / SR / SSR
5. 設計個體微調 → ±1–5 數值調整
6. 設計種族量表個人化 → 自訂觸發/效果
7. 設計被動技能 → 1–2 個，反映角色特質
8. 設計主動技能 → 2–4 個，反映角色戰鬥方式
9. 設計終極技 → 1 個，角色的標誌性大招
```

**設計原則：**
- 同種族同職業的不同英雄，技能必須有本質差異
- 種族量表個人化不能違反框架基本規則（如血怒必須與血量/受傷相關）
- 英雄個體微調不可超出稀有度允許範圍
- 職業關鍵字是基礎，英雄可以在此基礎上擴展但不可違反

**同種族同職業多英雄範例（概念預覽）：**

| 英雄 | 種族 | 職業 | 差異化方向 |
|------|------|------|-----------|
| 軍團統帥 | 人類 | 指揮官 | 防禦型指揮：兵力越多越硬，堅守陣線 |
| ？（未來）| 人類 | 指揮官 | 進攻型指揮：兵力越多攻擊越強，全軍衝鋒 |
| ？（未來）| 人類 | 指揮官 | 消耗型指揮：犧牲兵力觸發強力效果，送死戰術 |

> 三位都是人類·指揮官，都用軍令量表和號令關鍵字，但被動/主動/終極技完全不同，玩法截然不同。
