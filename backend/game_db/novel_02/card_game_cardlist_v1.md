## 卡表閱讀指南

**費用**：消耗的魔力水晶數量。
**稀有度**：● 普通 / ●● 精良 / ●●● 稀有 / ★ 傳說。
**牌組限制**：● 最多 3 份 / ●● 最多 2 份 / ●●● 最多 1 份 / ★ 最多 1 份且全牌組最多 3 張傳說。
**兵力數值格式**：HP / ATK / DEF。
**關鍵字縮寫**：〔守護〕〔突進〕〔疾走〕〔必殺〕〔汲取〕〔威壓〕〔穿透〕〔入場曲〕〔謝幕曲〕

## 九、設計備註

### 種族卡與英雄技能的互動原則

種族卡被設計為與 **種族量表框架** 互動，而非與特定英雄技能互動。這確保：
- 同種族的任何英雄都能有效使用所有種族卡
- 不會出現「這張種族卡只對某個英雄有用」的情況
- 種族卡透過量表關鍵字（軍令/共鳴/爐火/靈蘊/血怒/神力殘響）與英雄產生連結

### 守護卡的分佈

守護是防禦體系的核心。各來源的守護兵力分佈：
- 通用：持盾衛兵(2費) / 戰地牧師(4費) / 精英禁衛(5費)
- 人類：聖殿騎士(5費) / 皇家禁衛軍團(6費) / 堅守陣型法術（全兵力臨時守護）
- 矮人：自動哨兵器具(3費)
- 妖族：百鬼夜行的幻影（人形限定）/ 幻術師職業的免費幻影
- 半神族：信徒(2費) / 不朽神使(5費)
- 精靈/獸族：無種族專屬守護兵力（需依賴通用守護卡或職業機制）
- 中立：大陸浮游鯨(8費)

> 精靈和獸族缺乏守護是有意的設計——精靈靠法術控制代替守護，獸族靠進攻壓制讓對方沒機會打臉。

### 卡牌擴展方向

核心系列建立基礎卡池後，未來擴展系列可以加入：
- **英雄專屬卡**：與特定英雄技能產生協同的卡牌（如「軍團統帥的軍旗」）
- **跨種族卡**：需要特定種族+職業組合才能使用的稀有卡
- **天象卡**：可主動觸發天象事件的特殊法術
- **魔族卡**：僅敵方可使用的卡牌，擴展 Boss 戰多樣性
