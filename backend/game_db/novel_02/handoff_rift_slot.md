## 3. 現有「次元壁穩定度」機制（v3.2 現狀）

| 穩定度 | 效果 |
|--------|------|
| 100（初始） | 無效果 |
| ≤ 75 | 我方兵力欄位 -1 |
| ≤ 50 | 敵方每回合額外部署 1 個兵力 |
| ≤ 25 | 我方治療效果減半 |
| 0 | **直接敗北** |

> 內戰模式停用此機制。

**現有相關卡牌**：
- `S08 次元修補`（3 費法術）：穩定度 +15
- `F08 次元裂縫`（5 費場地·傳說）：穩定度每回合 -5，但所有法術/行動傷害 +50%
- `N05 次元壁碎片`（3 費中立傳說）：穩定度 +25
- `A10 星落之劍`（7 費傳說行動）：穩定度 ≤ 50 時額外 +15 傷害

問題：穩定度只是「越高越好」的單向資源，玩家沒有「主動降低」的動機。

## 4. 設計核心方向（claude.ai session 已確認）

### 4.1 引入「次元滲透裂縫」欄位

當穩定度 **< 50**，戰場中央出現一個特殊欄位「次元滲透裂縫」（不佔用原本的 5 個兵力欄）。

### 4.2 雙刃用法

| 使用方 | 可在裂縫做什麼 |
|--------|----------------|
| **敵方** | 從裂縫直接「滲透」部署黑暗單位（不消耗對方資源/不占其兵力欄） |
| **玩家** | 透過特定卡牌「佔據」裂縫，獲得強力 buff（佔據期間敵方無法滲透） |

### 4.3 玩家戰略張力

玩家現在面對兩個方向相反的目標：
- **守序派**：努力維持穩定度 ≥ 50（保安全）
- **混沌派**：主動讓穩定度下降到 < 50，搶在敵方之前佔據裂縫

這呼應 novel_02 世界觀中「冒險者」與「次元滲透」的本質——黑暗次元的力量是雙面刃。

## 5. 需要與使用者討論並確定的細節

接手後請主動引導討論以下問題（建議用 `ask_user_input_v0` 或類似的結構化提問，**每次最多 2–3 題，避免一次拋太多**）：

### A. 裂縫出現/消失的觸發

1. 穩定度具體門檻（建議 50，但是否要分階段？例如 <75 出現「裂痕」、<50 變成「裂縫」、<25 變成「次元門」？）
2. 穩定度回升超過門檻時，裂縫是否消失？正在被佔據時呢？
3. 是否允許場上同時存在多個裂縫（穩定度極低時）？

### B. 敵方滲透行為

1. 滲透頻率（每回合？每兩回合？由 AI 意圖預告？）
2. 滲透單位的設計（新增「滲透體」類別兵力卡池？或從現有 Boss 卡池抽取？）
3. 滲透單位被擊殺後是否影響穩定度（建議：擊殺敵方滲透體 → 穩定度 +5）

### C. 玩家佔據裂縫

1. 佔據方法（消耗特定卡牌？需要犧牲己方兵力？花費鬥志？）
2. 佔據後的具體 buff（種族量表加倍？魔力上限 +1？抽 1 張黑暗卡？英雄獲得新關鍵字？）
3. 佔據可維持幾回合？敵方能否搶回？

### D. 新增卡牌的範圍

是否要新增一批與裂縫互動的卡牌？建議至少：
1. 1 張「主動佔據」法術
2. 1–2 張「敵方滲透體」兵力卡（給敵方 AI 使用）
3. 1 張通用「裂縫共鳴」效果裝備或場地
4. 重新評估 F08 次元裂縫的數值是否需要調整

### E. 平衡與規則邊界

1. 內戰模式如何處理？（v3.2 內戰模式停用次元壁，此機制是否也停用？或改為雙方共享裂縫？）
2. 巢穴戰中是否啟用？（巢穴本身會降穩定度，可能很快觸發裂縫）
3. 試煉塔層數對裂縫機制的調整？
