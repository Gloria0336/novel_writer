## 驗收重點

- `water` 無 `bridge` / `ford` 時不可通行。
- `road` 會降低移動成本。
- `bridge` 讓水域可進入並套用橋成本。
- `secret_path` 對魔物軍隊有額外行軍優勢。
- `wall` / `moat` 對敵方進入有額外成本，為 M3/M5 施工與戰術地形保留鉤子。
- `ArmyMove` 必須從 army 當前位置開始，且每一步必須是相鄰 hex。
- 超出 `movement_points` 的命令會被拒絕。
- 軍隊可在前端被選取，並即時顯示可達範圍與路徑。

## 剩餘風險與後續銜接

- M2 只建立「戰略交戰觸發」，尚未展開成 M4 的 `TacticalBattle` / `BattleReplay`。
- `engine/turn.py` 的 combat 仍是 M1 placeholder，正式替換點仍保留在 M4。
- M3 施工完成後，`road` / `bridge` / `wall` / `moat` / `terraform` 對移動與交戰的影響已具備可接入的 cost / feature 位置。

## 最終判定

M2 驗收通過，可進入 M3「施工與地形改造」。
