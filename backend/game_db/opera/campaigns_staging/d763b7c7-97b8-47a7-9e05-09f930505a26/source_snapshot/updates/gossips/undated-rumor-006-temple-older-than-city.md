# 山頂神殿比城市更早存在

## 傳聞內容

城市裡流傳著一種很廣、也很容易被相信的說法：

> 城市是後來圍著深淵蓋起來的，但神殿早就在那裡。

這條傳聞某種程度上已經不是純八卦，而是很多人默認的「也許是真的」。

## 已知線索

- 神殿的存在感太古老，讓人很難把它視為普通城市建築。
- 孤山本身過於異常，強化了神殿早於城市的印象。
- 僧侶的態度像是在守著什麼，而不是單純住在那裡。
- 尚待確認：城市建立前是否已有神殿建築、宗教團體或相關遺跡。

## 可能影響

- 若為真，山頂神殿可能掌握深淵開發以前的歷史。
- 神殿的權威可能不只是宗教權威，也可能來自它與深淵起源的關係。
- 城市居民對神殿的敬畏會更合理，也更難被公會或商會取代。
- 若被證實為假，也能顯示神殿形象如何被民間想像放大。

- 若確認為正史，應轉寫到 `regions/` 的孤山或城市起源檔案，以及神殿相關派系檔案。
- 若維持傳聞狀態，可在居民談話與酒館敘事中反覆出現。
