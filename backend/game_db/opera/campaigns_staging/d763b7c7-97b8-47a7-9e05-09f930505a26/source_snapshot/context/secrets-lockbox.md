# Secrets Lockbox

> [!IMPORTANT]
> 本檔只給作者與 AI 參考，用於保存未揭露真相、後台秘密、深淵核心謎團、勢力黑箱與角色不知道的情報。公開檔可以引用秘密編號，但不要直接揭露內容。

## 使用規則

- 每個秘密使用固定 ID，例如 `secret-abyss-001`。
- 公開檔只寫「秘密引用：secret-abyss-001」，不寫秘密內容。
- 若秘密已在正文正式揭露，將狀態改為 `revealed`，並把公開部分整理進 `bible/worldbuilding.md` 或相關檔案。
- 角色知道的情報與作者知道的真相要分開記錄。
