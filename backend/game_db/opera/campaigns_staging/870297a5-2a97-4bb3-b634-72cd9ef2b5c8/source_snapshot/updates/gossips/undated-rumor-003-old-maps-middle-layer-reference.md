# 舊地圖在中層只能當參考

- `Rumor ID`: `rumor-003`
- `First Heard`: `unknown`
- `Location`: 深淵中層區
- `Source`: 中層探索隊、資深冒險者、地圖販售者
- `Reliability`: `high`
- `Status`: `unverified`
- `Related`: 中層區、舊地圖、溶洞、礦坑、地下湖泊、劇毒生態域、靈氣異常

## 傳聞內容

所有有經驗的冒險者都知道：舊地圖在中層只能當參考，不能當保證。

中層區的地形、路線與危險點，無法像地表道路那樣被視為長期固定。地圖仍然有用，因為它至少能提供大方向與已知標記；但若完全依賴舊地圖行動，很可能死得很快。

真正成熟的隊伍，從來不是「照圖走」，而是帶著地圖，再根據現場狀況即時修正。

## 已知線索

- 大型溶洞交錯區、礦坑分支區與地下湖泊周邊，常被認為最容易讓舊地圖失準。
- 劇毒生態域邊界與靈氣異常活躍區，也被列為高變動風險範圍。
- 地圖仍具備價值，但不能取代現場偵察與隊伍判斷。
- 尚待確認：中層地形變動的原因是自然坍塌、生態活動、靈氣異常，還是深淵本身的活性。

## 可能影響

- 探索隊需要攜帶地圖，但不能只依賴地圖。
- 地圖販售、更新與偵察情報可能形成重要產業。
- 使用過期地圖可能成為冒險事故、失蹤事件或陰謀陷害的原因。
- 若某張舊地圖仍準確，反而可能暗示該區域穩定或被人維護。

## 後續處理

- 可在 `regions/` 的中層區地理資料中補充地形變動規則。
- 若確認為中層核心設定，應轉寫到區域檔案或世界觀檔案。
- 若保留為冒險者經驗法則，可在任務準備、採購與隊伍爭論場景中引用。
