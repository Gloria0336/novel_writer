# 三途川不是整段都安全

- `Rumor ID`: `rumor-002`
- `First Heard`: `unknown`
- `Location`: 三途川、深淵淺層區
- `Source`: 冒險者圈、採集隊、管理方標定告示
- `Reliability`: `high`
- `Status`: `unverified`
- `Related`: 三途川、淺層區、公會、管理方、安全區、變動水域

## 傳聞內容

雖然三途川位於淺層區，且屬於公開開發範圍，但它並不是整段地下河流都可自由活動。

目前已知三途川內部仍劃分有安全區、非安全區，以及需要額外警戒的變動水域。對一般冒險者而言，真正可靠的只有公會與管理方當前標定的安全區。

冒險者之間常說：

> 三途川能走，但不能亂走。

## 已知線索

- 離開安全區之後，水位可能突然變化，水流也可能改道。
- 視野惡化、水中生物活動增強、通道與岸線環境不穩定，都是常被提及的風險。
- 三途川雖然被歸入淺層公開範圍，但仍保留內部風險分級。
- 尚待確認：目前安全區、非安全區與變動水域的實際邊界。

## 可能影響

- 一般冒險者若誤把三途川全段視為安全，可能導致事故。
- 公會與管理方的即時標定會成為角色行動的重要依據。
- 三途川可同時作為低門檻探索場景與突發危機場景。
- 若標定失準，可能引發公會責任、商會損失或民間恐慌。
