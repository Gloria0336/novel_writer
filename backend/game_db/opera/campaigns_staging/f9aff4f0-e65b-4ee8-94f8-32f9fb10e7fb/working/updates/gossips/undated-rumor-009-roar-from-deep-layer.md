# 有人在中層底部聽見來自深層的怪物吼叫

- `Reliability`: `low`
- `Status`: `unverified`

## 傳聞內容

傳聞說，曾有人在中層區底部活動時，聽見從更深處傳來巨大、低沉、無法判斷來源的吼叫聲。

那聲音不像已知靈獸，也不像任何普通怪物，更不像地形回音。

所以這條話才會被一再重述：

> 中層底部下面，真的有東西在叫。

這種傳聞最可怕的地方在於，它不像純神話，而像是某種可能發生過的經驗殘片。

## 已知線索

- 傳聞帶有明確場景：中層底部、來自更深處、低沉且巨大。
- 聲音被描述為不像已知靈獸或普通怪物，暗示目擊者有一定經驗。
- 也可能只是地形回音、靈氣異常、水流震動或恐懼造成的誤判。
- 尚待確認：是否存在可靠倖存者、探索日誌或多方一致證詞。

## 可能影響

- 若為真，可能支持「深層區存在異常強大物種」的相關傳聞。
- 即使為假，也會加劇冒險者對中層底部與深層入口的恐懼。
- 這類聲音事件可作為探索隊撤退、封鎖或公會調查的導火線。
- 若多次出現類似報告，可能迫使公會重新評估中層底部風險。
