# Calendar

> 本檔定義深淵城市的輕量劇中曆法。初期只要求能穩定追蹤事件，月份名稱、節慶與商業週期可等故事需要時再擴充。

## 日期格式

目前採用：

```text
深淵曆 Y000-M01-D01
```

- `Y000`: 故事起始年份占位。
- `M01`: 月份或城市記錄週期。
- `D01`: 當月日期。

每日更新檔使用同樣格式命名：

```text
updates/daily/Y000-M01-D01.md
```

## 使用規則

- 正文尚未確定正式曆法前，所有事件先使用 `深淵曆 Y000-M01-D01`。
- 同一天內可用「清晨、上午、午後、傍晚、夜間、深夜」標記事件順序。
- 若同一天事件密度很高，可在日期檔內新增時間段，不另拆檔。
- 若日後建立正式月份名稱，保留檔名格式不變，在檔內補充顯示名稱。

## 待定項目

- 年號來源
- 月份名稱
- 一年幾個月
- 一月幾日
- 公會結算日
- 商會拍賣或交割週期
- 深淵封鎖檢查日
- 城市節慶與紀念日
