# Character

> 本檔是群像角色總檔。格式參考其他小說專案的 `characters.yaml`，但改用 Markdown 方便補充段落、表格與狀態筆記。

## 使用規則

- 角色正式登場或確定會牽動劇情時，才加入本檔。
- 每個角色都要追蹤「目前所在區域、知道什麼、想要什麼、被哪條事件線牽動」。
- 未揭露真相不要直接寫在公開欄位，請使用 `context/secrets-lockbox.md` 的秘密編號。
- 若角色數量過多，先保留本檔作索引，再按角色拆分子檔。
